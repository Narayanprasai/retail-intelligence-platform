# Retail Intelligence Platform

End-to-end AWS data platform for Australian retail analytics and ML.

## Stack
- **Ingestion:** Config-driven Python + AWS S3
- **Processing:** AWS Glue + PySpark
- **Warehouse:** Redshift Serverless + dbt
- **ML:** SageMaker Feature Store + XGBoost
- **Orchestration:** Apache Airflow (MWAA)
- **Serving:** FastAPI + AWS Lambda + API Gateway
- **Dashboard:** AWS QuickSight
- **IaC:** Terraform
- **CI/CD:** GitHub Actions
